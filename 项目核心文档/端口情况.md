方式1: 检查端口（最常用）
cd ~/Desktop/CS/创业/相亲项目开发/幸福力项目开发./scripts/port-check.sh check

输出示例:
=== 幸福力项目端口检查工具 ===检查 后端服务 (端口 3000)...✗ 端口 3000 被占用COMMAND   PID   USER   ...node      38743 cs     ...检查 前台H5 (端口 5173)...✓ 端口 5173 空闲

方式2: 清理端口
# 清理所有项目端口./scripts/port-check.sh kill# 只清理后端端口./scripts/port-check.sh kill 3000

方式3: 一键启动（最省心）
./scripts/start-all.sh


常用命令速查
场景	命令
检查所有端口	./scripts/port-check.sh check
清理所有端口	./scripts/port-check.sh kill
清理指定端口	./scripts/port-check.sh kill 3000
一键启动项目	./scripts/start-all.sh
查看Node进程	ps aux \| grep node
手动清理3000端口	lsof -ti:3000 \| xargs kill -9
