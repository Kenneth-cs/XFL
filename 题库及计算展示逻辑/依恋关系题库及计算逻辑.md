## 二、核心规则说明（代码逻辑可直接映射）
### 1. 计分规则
- 每题性质：多选题（选中计1分，未选中计0分）
- 得分计算：
  - 焦虑测试得分（A）= 测试一中选中题目的数量总和（范围：0-12分）
  - 回避测试得分（B）= 测试二中选中题目的数量总和（范围：0-12分）
  - 安全感测试得分（C）= 测试三中选中题目的数量总和（范围：0-12分）

### 2. 依恋类型判定逻辑（优先级：紊乱型 > 焦虑型 > 回避型 > 安全型）
```python
# 伪代码示例
def get_attachment_type(A, B, C):
    if A >= 5 and B >= 5:
        return "高焦虑+高回避（紊乱型）"
    elif A >= 5:
        return "焦虑型"
    elif B >= 5:
        return "回避型"
    elif C >= 5:
        return "安全型"
    else:
        # 未达任何类型判定阈值，按默认提示
        return "待进一步沟通（请联系红娘老师）"
```

### 3. 二维度映射规则（学界核心逻辑适配）
| 维度         | 高维度判定条件 | 低维度判定条件 | 维度核心定义 |
|--------------|----------------|----------------|--------------|
| 焦虑型维度   | A ≥ 5          | A < 5          | 衡量对「关系被抛弃、不被爱」的担忧程度 |
| 回避型维度   | B ≥ 5          | B < 5          | 衡量对「情感亲密、过度依赖」的抗拒程度 |

### 4. 结果输出规则
- 前台展示文案：`您在《幸福力婚恋关系测试》中显示是：{attachment_type}。详细测试结果请与红娘老师沟通。`
